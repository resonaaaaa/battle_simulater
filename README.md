## 项目简介
这是一个基于Python的角色对战系统，允许用户创建角色并进行战斗
本系统有多个可选角色，每个角色都有独特的技能和属性
点击运行文件夹中的battle_simulator.exe即可体验角色对战!

## 自定义
- 你可以通过编辑`battle_character.py`文件来创建和修改角色
- 每个角色都继承自`Character`类，你可以重写其中的方法来实现不同的技能和效果，创建自定义角色。
- 在修改'battle_character.py'后，请记得更新用于实现GUI的'battle_gui.py'文件中的角色列表，以便在界面上选择新角色。
- 完成上述工作后，运行用于创建exe文件的'build_exe.bat'文件来生成新的exe文件。

## 角色介绍
### 1.狂战士Berserker
*痴迷于战斗的勇士，愤怒会让他变得更强。正常人都不想和他并肩作战，因为他上头时敌我不分！*
**技能**：
- 1. 狂怒：增加攻击力，但降低防御力，持续3回合，需要怒气值50
- 2. 怒焰：释放所有怒气，对敌人造成大量伤害，伤害由怒气值与攻击力计算,仅仅在狂怒状态可使用，且需至少20点怒气值
- 3. 狂暴重击：消耗20点怒气值，对敌人造成部分穿透防御的伤害，并有概率使敌人防御力下降2回合，冷却4回合

### 2.吸血鬼Vampire
*神秘而强大，能从敌人身上吸取生命。自从现代医学建立了血库之后，他的生活质量有了显著提升！*
**技能**：
- 1. 吸血：攻击敌人时回复自身生命值
- 2. 蝙蝠召唤：召唤蝙蝠，在自身发动攻击后对敌人造成额外伤害，蝙蝠上限为3只
- 3. 引爆蝙蝠：牺牲所有蝙蝠，对敌人造成伤害，蝙蝠越多伤害越高

### 3.骑士Knight
*忠诚的守护者，擅长防守和反击。他已经在战场上被打快三十年了……*
**技能**：
- 1. 格挡反击：概率格挡一次攻击，成功后反击造成伤害，冷却3回合
- 2. 神圣冲击：根据自身已损失的生命值造成伤害，损失越多伤害越高，每次战斗只能使用一次

### 4.龙人DragonHuman
*半人半龙的战士，拥有强大的龙息和火焰能力。他的鳞片看起来很有质感？那是因为他经常打蜡！*
**技能**：
- 1. 龙息：对敌人造成部分穿透防御的伤害,并短暂提高自身的攻击力
- 2. 龙炎：对敌人造成最大生命值的百分比伤害，附带龙炎效果，持续2回合，龙炎会对敌人造成持续伤害，并使得敌人无法回复生命值

### 5.烈焰女巫FlameWitch
*掌控火焰的女巫，能用火焰保护自己和攻击敌人。她的厨艺真的很高超，尤其是烧烤！*
**技能**：
- 1. 烈焰护盾：生成一个护盾，吸收一定量的伤害，护盾被打破时反弹部分伤害给攻击者
- 2. 爆炎冲击：对敌人造成大量火焰伤害，伤害无视防御，附带灼烧效果，持续3回合，灼烧效果每回合造成伤害并降低敌人攻击力

### 6.人鱼Mermaid
*拥有美丽歌喉的水中生灵，可以操控水流进行防御。不要让她上岸，除非你想听到……*
**技能**：
- 1. 水流护盾：生成一个水流护盾，吸收一定量的伤害,护盾值根据角色等级和防御属性提升，冷却5回合
- 2. 治愈歌谣：唱出治愈的歌谣，按照已损失生命值的百分比回复自身生命值，每次战斗限2次
- 3. 宁静波纹：消除自身所有的debuff，每次战斗限2次
- 4. 激流：对敌人造成伤害并使其防御力下降，持续2回合，冷却4回合

### 7.狼人Wolfman
*夜晚的猎人，可以借用先祖的原始力量。讨厌吃素，因此消化不大好*
**技能**：
- 1. 原始进化：当生命值低于40%时，进入返祖状态，攻击将部分穿透敌人的防御
- 2. 血腥之咬：对敌人造成伤害，并根据造成的伤害回复自身生命值
- 3. 狼嚎：仅在返祖状态可用，发出狼嚎，降低敌人的攻击力和防御力，持续2回合，冷却4回合

### 8.德鲁伊Druid
*与自然融为一体的守护者，能召唤自然力量进行战斗，但他的前一份工作其实在动物园*
**技能**：
- 1. 自然召唤：随机召唤一个自然生物来协助战斗，每个生物有不同的能力，上限2个
    - 熊：高额攻击力
    - 鹰：高额暴击率
    - 树人：代替德鲁伊承受伤害
    - 兔子：使得德鲁伊概率闪避攻击
    - 鹿：德鲁伊发动攻击时有概率回复生命值
- 2.树灵佑护：暂时提高自身的防御力，持续3回合,冷却5回合，结束后少量回复生命值
### 9.雪精灵SnowElf
*来自寒冷北方的神秘生物，她总以为自己很会讲笑话*
**技能**：
- 1. 冰晶箭矢：使得自己的攻击变为冰霜箭矢，伤害提高，并且每次攻击有30%概率冻结对方1回合(每次使用技能仅触发一次冰冻)，冻结状态下无法行动。该效果能生效2次attack，冷却4回合。
- 2. 暴风雪：限定技，每次战斗仅能使用1次，造成持续伤害，在暴风雪生效期间，敌人攻击力下降20%，持续3回合
- 3. 寒霜长矛: 造成部分穿透防御的伤害，且有70%概率使目标冻结1回合，冷却4回合
