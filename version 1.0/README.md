## 项目简介
这是一个基于Python的角色对战系统，允许用户创建角色并进行战斗
本系统有多个可选角色，每个角色都有独特的技能和属性
点击运行压缩包中的battle_simulator.exe即可体验角色对战!

## 角色介绍
### 1.狂战士Berserker
*痴迷于战斗的勇士，愤怒会让他变得更强*
**技能**：
- 1. 狂怒：增加攻击力，但降低防御力
- 2. 怒焰：释放怒气，对敌人造成大量伤害

### 2.骑士Knight
*忠诚的守护者，擅长防守和反击*
**技能**：
- 1. 盾牌防御：概率格挡一次攻击
- 2. 神圣冲击：根据自身已损失的生命值造成伤害，损失越多伤害越高
